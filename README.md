# Midtown Podcast Studio Website

This is a simple one-page podcast studio rental website.

## How to edit it

Open `index.html` to change:
- Studio name
- Headlines
- Services
- Pricing
- FAQ
- Email / Calendly link

Open `styles.css` to change:
- Colors
- Spacing
- Fonts
- Layout

## How to preview it

Double-click `index.html` and it should open in your browser.

## How to put it online

1. Create a free Vercel account.
2. Create a free GitHub account.
3. Upload these files to a GitHub repository.
4. In Vercel, click New Project.
5. Import your GitHub repository.
6. Click Deploy.
7. Buy a domain and connect it in Vercel.

You can replace photo placeholders later with real studio photos.
